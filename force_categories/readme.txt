Displays 4 news stories (that have a particular category) with various sizes.

== Description ==

First story shows large picture, title and extended

Second story shows thumbnail, title and extended

3-5 show headline only

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
