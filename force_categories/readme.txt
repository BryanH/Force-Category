=== Force Categories ===
Contributors:
Tags: post, categories, user, author, contributor
Requires at least: 3.0.0
Tested up to: 3.0.1
Stable tag: git://github.com/BryanH/Force-Category.git

When an author or contributor saves a post, this will forces or exclude specific categories.

== Description ==

Every post by a specific author can be:

1. Forced to belong to one or more specified categories
1. Prevented from belonging to one or more specified categories

== Installation ==

1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation

2. Activate the Plugin from Plugins page.
